# times-monguchi

個人用リポジトリの初期形です。**Cursor** と **Claude Code** の両方で同じ方針を共有できるよう、`AGENTS.md` と `CLAUDE.md`、`.cursor/`、`.claude/` を備えています。

## 使い方

| 読むもの | 内容 |
|----------|------|
| [AGENTS.md](./AGENTS.md) | エージェント共通ルール（最優先の方針） |
| [CLAUDE.md](./CLAUDE.md) | Claude Code 用の短いエントリポイント |

## ディレクトリ

- `.cursor/rules/` — Cursor 用ルール（`.mdc`）
- `.cursor/skills/` — Cursor 用スキル
- `.claude/settings.json` — Claude Code の共有設定（権限など）

スタックを決めたら、この `README.md` にセットアップ手順とコマンドを書き足してください。
