# Claude Code — プロジェクトメモリ

Claude Code 向けの起動時コンテキスト。**Cursor と併用**するため、詳細な共通方針は `AGENTS.md` に集約している。

## 必ず読む

- **`AGENTS.md`** — コミュニケーション言語、優先順位、セキュリティ、作業のルール

## このリポジトリの設定

- **共有設定**: `.claude/settings.json`（権限のベースラインなど）
- **個人設定**: `.claude/settings.local.json`（Git に含めない。API や個人好みの上書き用）
- **Cursor 側**: `.cursor/rules/` と `.cursor/skills/` — 実装パターンやワークフローはここも参照して一貫性を保つ

## 作業ルール（要約）

- ユーザーの指示の範囲内で最小限の変更にとどまる。
- `.env` やシークレットを読み取らない・出力に含めない（必要なら `.env.example` のみ）。
- 説明は簡潔に。
